# Panel（提示词选择面板）

### 打开面板

- **触发方式**：在任意输入框中输入触发词（如 `/`）
- **显示位置**：屏幕中央上方 5% 的位置
- **外观**：带有阴影的白色卡片，带有搜索框和提示词列表

### 面板内容结构

```
┌─────────────────────────────────────┐
│ [搜索输入框]                         │
├─────────────────────────────────────┤
│ Recent                              │  ← 有最近使用时显示
│   - Prompt Item 1                   │
│   - Prompt Item 2                   │
├─────────────────────────────────────┤
│ All Prompts                         │
│   - Prompt Item A                   │  ← 键盘导航高亮显示
│   - Prompt Item B                   │
└─────────────────────────────────────┘
```

### 键盘导航

| 按键        | 功能                     |
| ----------- | ------------------------ |
| `ArrowDown` | 向下选择下一个提示词     |
| `ArrowUp`   | 向上选择上一个提示词     |
| `Enter`     | 确认选择当前高亮的提示词 |
| `Escape`    | 关闭面板，返回输入框     |

### 鼠标交互

- **悬停**：鼠标悬停在提示词上时高亮显示
- **点击**：点击提示词选中并插入

### 搜索功能

- 输入文字实时过滤提示词
- 搜索范围：标题、描述、标签
- 搜索时清空最近使用列表，显示匹配结果

### 选中状态

- 当前选中的提示词添加 `selected` 类
- 高亮样式：背景色变化或边框
