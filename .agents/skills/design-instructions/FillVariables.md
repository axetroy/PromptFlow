# Fill Variables（变量填写弹窗）

### 打开条件

- 选中的提示词包含变量（`<VAR name="xxx">`）
- 点击提示词后自动弹出

### 弹窗结构

```
┌─────────────────────────────────────┐
│ 💬 Fill Variables                   │  ← 标题栏（渐变背景）
│ Prompt Title                        │  ← 副标题
├─────────────────────────────────────┤
│                                     │
│ Variables                           │  ← 变量列表
│ ┌─────────────────────────────────┐ │
│ │ "name" [default]                │ │
│ │ 描述文字                         │ │
│ │ ┌─────────────────────────────┐ │ │
│ │ │ [输入框]                    │ │ │  ← 页面打开聚焦第一个未填写的字段
│ │ └─────────────────────────────┘ │ │
│ └─────────────────────────────────┘ │
│                                     │
│ Preview                    [📋复制] │  ← 实时预览
│ ┌─────────────────────────────────┐ │
│ │ 这里显示填充后的内容             │ │
│ │ <VAR>会被高亮显示               │ │
│ └─────────────────────────────────┘ │
├─────────────────────────────────────┤
│ 💡 Tab/Enter 下一个 · Esc 取消  [取消] [确认] │
└─────────────────────────────────────┘
```

### 变量输入行为

| 行为       | 说明                       |
| ---------- | -------------------------- |
| 默认值     | 有默认值时自动填充         |
| Tab 键     | 跳转到下一个输入框         |
| Enter 键   | 跳转到下一个输入框（单行） |
| Ctrl+Enter | 提交表单                   |

### 关闭方式

| 方式         | 操作                       |
| ------------ | -------------------------- |
| ESC 键       | 取消并关闭，恢复输入框焦点 |
| 点击取消按钮 | 取消并关闭                 |
| 点击遮罩层   | 取消并关闭                 |

### 预览功能

- 实时显示填充后的内容
- 变量名高亮显示
- 未填写的必填变量用特殊样式提示

### 复制功能

- 点击复制按钮可复制预览内容
- 复制成功显示 "已复制" 提示
