# Prompt Preview（提示词预览弹窗）

### 打开方式

- 在设置页面的提示词表格中，点击行内 👁 按钮打开

### 弹窗结构

```
┌──────────────────────────────────────────────────┐
│ 📄 标题 [标签]                              [×] │
│                                                   │
│ 描述文字                                           │
│                                                   │
│ [tag1] [tag2] [tag3]                               │
│                                                   │
│ ┌─────────────────────────────────────────────┐   │
│ │ 提示词内容，变量会高亮显示                   │   │
│ │ 比如 [variable_name] 以紫色背景展示           │   │
│ └─────────────────────────────────────────────┘   │
│                                                   │
│ Last updated: 2024-01-15 10:30                    │
└──────────────────────────────────────────────────┘
```

### 显示内容

| 区域 | 说明 |
|------|------|
| 标题栏 | 图标 + 标题 + 类型标签（Default/Custom/Synced） |
| 描述 | 可选，灰色文字显示 |
| 标签 | 所有标签以 Ant Design Tag 显示 |
| 内容 | 等宽字体代码块，变量以紫色高亮显示 |
| 更新时间 | 底部灰色小字 |

### 变量高亮样式

| 属性 | 值 |
|------|-----|
| 背景色 | `#ede9fe` |
| 文字色 | `#7c3aed` |
| 下边框 | `2px solid #a78bfa` |
| 圆角 | 3px |
| 内边距 | 1px 4px |

### 类型标签颜色

| 类型 | 标签颜色 | 判断条件 |
|------|---------|---------|
| Default | blue | `prompt.isDefault` |
| Custom | green | 非 Default 且 id 不以 `sync-` 开头 |
| Synced | purple | id 以 `sync-` 开头 |

### 交互行为

- 弹窗无底部按钮（`footer={null}`）
- 点击遮罩层或右上角 × 关闭
- 宽度 700px，内容区域最大高度 60vh
