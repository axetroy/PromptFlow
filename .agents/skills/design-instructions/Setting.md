## General Settings（通用设置）

### Trigger Command（触发命令）

| 属性     | 说明                          |
| -------- | ----------------------------- |
| 功能     | 设置打开 PromptPanel 的触发词 |
| 默认值   | `/prompts`                    |
| 输入方式 | 文本输入框                    |
| 位置     | 页面顶部第一个卡片            |

### Auto Sync Interval（自动同步间隔）

| 属性   | 说明                                      |
| ------ | ----------------------------------------- |
| 功能   | 设置从 GitHub 仓库自动同步的频率          |
| 选项   | 每15分钟、每30分钟、每小时、每2小时、每天 |
| 默认值 | 每小时                                    |

## Prompts 列表（提示词管理）

### 提示词类型标签

| 标签颜色      | 类型    | 说明                   |
| ------------- | ------- | ---------------------- |
| 蓝色 (Blue)   | Default | 内置默认提示词         |
| 绿色 (Green)  | Custom  | 用户自定义提示词       |
| 紫色 (Purple) | Synced  | 从 GitHub 同步的提示词 |

### 表格列说明

| 列名        | 说明                   |
| ----------- | ---------------------- |
| Title       | 提示词标题，带类型标签 |
| Description | 描述文本，超长省略     |
| Tags        | 标签列表，可多行显示   |
| Status      | 启用/禁用开关          |
| Actions     | 操作按钮组             |

### 操作按钮

| 按钮    | 图标 | 功能           | 限制                         |
| ------- | ---- | -------------- | ---------------------------- |
| Preview | 👁   | 预览提示词内容 | 无                           |
| Edit    | ✏    | 编辑提示词     | 禁用：Default 和 Synced 类型 |
| Delete  | 🗑   | 删除提示词     | 禁用：Default 和 Synced 类型 |

### 顶部操作按钮

| 按钮             | 功能                     |
| ---------------- | ------------------------ |
| Sync from GitHub | 打开 GitHub 同步管理器   |
| Export           | 导出所有提示词为 JSON    |
| Import           | 导入 JSON 文件添加提示词 |
| Add Prompt       | 添加新提示词             |

## Add/Edit Prompt Modal（添加/编辑提示词弹窗）

### 打开方式

- 点击 "Add Prompt" 打开添加弹窗
- 点击行内 "Edit" 按钮打开编辑弹窗

### 弹窗结构

```
┌──────────────────────────────────────────────────────┐
│ Add New Prompt / Edit Prompt                        │
├──────────────────────────────────────────────────────┤
│                                                      │
│ Title *                                              │
│ [________________________________]                   │
│                                                      │
│ Content *                                            │
│ [                                                  ] │
│ [                                                  ] │
│ [                                                  ] │
│ [                                                  ] │
│ [                                                  ] │
│ [                                                  ] │
│                                                      │
│ Description                                          │
│ [________________________________]                   │
│                                                      │
│ Tags                                                 │
│ [________________________________]                   │
│ Tip: Press enter or comma to create tags            │
│                                                      │
│                    [Cancel]  [Create/Update]        │
└──────────────────────────────────────────────────────┘
```

### 表单字段

| 字段        | 类型     | 必填 | 说明                              |
| ----------- | -------- | ---- | --------------------------------- |
| Title       | 文本输入 | 是   | 提示词标题                        |
| Content     | 多行文本 | 是   | 提示词内容，支持 `<VAR>` 变量语法 |
| Description | 文本输入 | 否   | 提示词描述                        |
| Tags        | 标签输入 | 否   | 按回车或逗号创建标签              |

### 表单验证

- Title 为空时显示 "Please enter a title"
- Content 为空时显示 "Please enter content"

## Sync from GitHub（GitHub 同步管理器）

### 打开方式

点击 Prompts 卡片顶部的 "Sync from GitHub" 按钮

### 仓库信息显示

```
┌─────────────────────────────────────────────────────────┐
│ 🔄 Synced from GitHub                                     │
│                                                          │
│ [Green] Active  owner/repo • main                        │
│               Last sync: 2024-01-15 10:30               │
│               [Sync]                                     │
│                                                          │
│ [Default] Disabled  owner/repo • develop                 │
│               Last sync: Never                           │
│               [Sync]                                     │
└─────────────────────────────────────────────────────────┘
```

### 仓库状态

| 状态     | 颜色 | 说明                   |
| -------- | ---- | ---------------------- |
| Active   | 绿色 | 仓库已启用，同步进行中 |
| Disabled | 灰色 | 仓库已禁用             |

### 每个仓库的操作

| 操作     | 说明                                  |
| -------- | ------------------------------------- |
| Sync     | 手动同步该仓库                        |
| 仓库信息 | 显示 owner/repo、branch、最后同步时间 |

## Usage Statistics（使用统计）

### 表格列

| 列名             | 说明                                       |
| ---------------- | ------------------------------------------ |
| Prompt Title     | 提示词标题                                 |
| Usage Count      | 使用次数                                   |
| Popularity Score | 热度分数（基于使用次数和最近使用时间计算） |

### 热度分数计算

- **公式**: `score = N × (0.5)^(T / 14)`
- **N**: 使用次数
- **T**: 距离上次使用的天数
- **半衰期**: 14 天（每 14 天分数减半）

### 清空历史

- 点击 "Clear History" 按钮
- 显示确认弹窗："Clear usage history? This will reset all usage statistics. This action cannot be undone."

## Prompt Preview Modal（提示词预览弹窗）

### 打开方式

点击提示词行的 "Preview" 按钮（👁）

### 显示内容

- 标题
- 描述
- 内容（支持变量高亮）
- 标签
- 创建/更新时间

## Import/Export（导入导出）

### Export（导出）

- 点击 "Export" 按钮
- 下载包含所有提示词的 JSON 文件

### Import（导入）

- 点击 "Import" 按钮
- 选择 JSON 文件
- 提示词会被添加到现有列表中

## 错误处理

### 错误提示

- 使用 Ant Design 的 `message` 组件显示
- 显示在页面顶部
- 自动消失

### 确认弹窗

- 删除操作显示确认弹窗
- 必须点击确认才会执行删除
