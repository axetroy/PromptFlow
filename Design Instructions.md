# PromptFlow UI 交互指导书

## 一、Panel（提示词选择面板）

### 1.1 打开面板

- **触发方式**：在任意输入框中输入触发词（如 `/`）
- **显示位置**：屏幕中央上方 5% 的位置
- **外观**：带有阴影的白色卡片，带有搜索框和提示词列表

### 1.2 面板内容结构

```
┌─────────────────────────────────────┐
│ [搜索输入框]                         │
├─────────────────────────────────────┤
│ Recent                              │  ← 有最近使用时显示
│   - Prompt Item 1                   │
│   - Prompt Item 2                   │
├─────────────────────────────────────┤
│ All Prompts                         │
│   - Prompt Item A                   │  ← 键盘导航高亮显示
│   - Prompt Item B                   │
└─────────────────────────────────────┘
```

### 1.3 键盘导航

| 按键        | 功能                     |
| ----------- | ------------------------ |
| `ArrowDown` | 向下选择下一个提示词     |
| `ArrowUp`   | 向上选择上一个提示词     |
| `Enter`     | 确认选择当前高亮的提示词 |
| `Escape`    | 关闭面板，返回输入框     |

### 1.4 鼠标交互

- **悬停**：鼠标悬停在提示词上时高亮显示
- **点击**：点击提示词选中并插入

### 1.5 搜索功能

- 输入文字实时过滤提示词
- 搜索范围：标题、描述、标签
- 搜索时清空最近使用列表，显示匹配结果

### 1.6 选中状态

- 当前选中的提示词添加 `selected` 类
- 高亮样式：背景色变化或边框

## 二、Fill Variables（变量填写弹窗）

### 2.1 打开条件

- 选中的提示词包含变量（`<VAR name="xxx">`）
- 点击提示词后自动弹出

### 2.2 弹窗结构

```
┌─────────────────────────────────────┐
│ 💬 Fill Variables                   │  ← 标题栏（渐变背景）
│ Prompt Title                        │  ← 副标题
├─────────────────────────────────────┤
│                                     │
│ Variables                           │  ← 变量列表
│ ┌─────────────────────────────────┐ │
│ │ "name" [default]                │ │
│ │ 描述文字                         │ │
│ │ ┌─────────────────────────────┐ │ │
│ │ │ [输入框]                    │ │ │  ← 页面打开聚焦第一个未填写的字段
│ │ └─────────────────────────────┘ │ │
│ └─────────────────────────────────┘ │
│                                     │
│ Preview                    [📋复制] │  ← 实时预览
│ ┌─────────────────────────────────┐ │
│ │ 这里显示填充后的内容             │ │
│ │ <VAR>会被高亮显示               │ │
│ └─────────────────────────────────┘ │
├─────────────────────────────────────┤
│ 💡 Tab/Enter 下一个 · Esc 取消  [取消] [确认] │
└─────────────────────────────────────┘
```

### 2.3 变量输入行为

| 行为       | 说明                       |
| ---------- | -------------------------- |
| 默认值     | 有默认值时自动填充         |
| Tab 键     | 跳转到下一个输入框         |
| Enter 键   | 跳转到下一个输入框（单行） |
| Ctrl+Enter | 提交表单                   |

### 2.4 关闭方式

| 方式         | 操作                       |
| ------------ | -------------------------- |
| ESC 键       | 取消并关闭，恢复输入框焦点 |
| 点击取消按钮 | 取消并关闭                 |
| 点击遮罩层   | 取消并关闭                 |

### 2.5 预览功能

- 实时显示填充后的内容
- 变量名高亮显示
- 未填写的必填变量用特殊样式提示

### 2.6 复制功能

- 点击复制按钮可复制预览内容
- 复制成功显示 "已复制" 提示

## 三、主题支持

### 3.1 自动主题切换

- 根据系统 `prefers-color-scheme` 自动切换
- 亮色模式：白色背景
- 暗色模式：深色背景

### 3.2 面板暗色模式

- 面板容器添加 `dark` 类时使用暗色样式

---

## 四、状态管理

### 4.1 Panel 状态

- 是否打开
- 当前选中的索引
- 搜索文本
- 最近使用的提示词 ID

### 4.2 VariableModal 状态

- 待填充的提示词
- 变量名到值的映射

### 4.3 焦点管理

- 打开面板时，搜索框自动获得焦点
- 关闭面板时，恢复原始输入框焦点和光标位置

---

## 五、注意事项

1. **Shadow DOM 隔离**：Panel 和 VariableModal 使用 Shadow DOM，样式完全独立
2. **事件冒泡**：Shadow DOM 内的事件需要特殊处理才能传播到外部
3. **键盘事件**：全局键盘事件在 content.ts 中处理，不在 React 组件内部
4. **点击外部关闭**：点击面板/弹窗外部区域应关闭面板
